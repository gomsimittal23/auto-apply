 Job Description 

3+ Yrs of Experience in Full Stack Engineers who have hands-on experience building real, complex
applications using agent engineering tools such as Claude Code, OpenAI Codex, or similar AI-native
development platforms.
This role is for engineers who can work across business, product, and technology teams to automate
manual workflows, build internal productivity tools, and create AI-powered features for end users.
Must-Have Experience
Real application development using agent engineering tools
The candidate must have used tools such as Claude Code, Codex, Cursor, or similar platforms to
build production-grade or complex real-world applications — not just prototypes or experiments.
Hands-on GenAI application development
The candidate must have built AI agents using GenAI APIs to solve complex end-user requirements
or automate complex internal workflows across multiple teams.


Key Responsibilities

Build full-stack applications using modern frontend, backend, API, and database technologies.
Automate manual operational workflows across business functions using AI and software.
Build AI agents, copilots, workflow automation tools, and end-user AI features.
Integrate GenAI platforms with internal systems, SaaS tools, APIs, and databases.
Use AI-native engineering tools across the SDLC for development, testing, debugging,
documentation, and deployment.
Convert ambiguous business problems into practical, scalable, production-ready solutions.
Ideal Candidate
A hands-on full-stack engineer who has already shipped meaningful applications using AI-native
engineering tools and GenAI APIs, and can apply them to solve real business and user problems.
